# MLB Game Prediction

This project predicts MLB game outcomes using machine learning.

## Dataset
Kaggle MLB dataset (~2400 games, 20+ features)

## Models
- Logistic Regression
- Random Forest

## Features
- Elo difference
- Rating difference
- Pitcher difference
- Probability difference

## Results
Random Forest performed better than Logistic Regression.

## How to Run
Upload dataset to Google Colab and run all cells.
